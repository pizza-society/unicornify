## Requirements

* Python 3.10+
* pipenv

## Getting started

Install dependencies

```bash
  pipenv install
```

Get into the virtual environment

```bash
  pipenv shell
```

Run the live server

```bash
uvicorn app.main:app --reload
```

## Running tests

To start running tests, run this command in virtualenv:

```bash
pytest -v
```
